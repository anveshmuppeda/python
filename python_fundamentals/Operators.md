**Operators are used to compair two values these is know as Comparison Operators**  
- Equal -> ==  
```console  
$ print(2==2)  
True   
```  
- Not Equal -> !=  
```console  
$ print(2!=4)  
True 
```   
- Greaterthan -> >    
```console  
$ print(4>2)  
True 
```  
- Greaterthan (or) Equal -> >=   
```console  
$ print(4>=2)  
True 
```  
- Smallerthan -> <    
```console  
$ print(2<5)  
True 
```  
- Smallerthan (or) Equal -> <=    
```console  
$ print(2<=7)  
True 
```
