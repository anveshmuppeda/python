# Literals
 The data elements with fixed values that are stored in the source code and do not change during a program's execution 
 Literals are of four types
 1. **Integers**
 Examples
 ``` 12,2,3,232,-10,1112,21```
 1. - Octal numbers 

## examples  

| 2 | 1 | 0 |  
| -- | -- | -- |  
| 8^2 | 8^1 | 8^0 |  
| 1 | 2 | 3 |  
|64|16|3|  
|total=83|  


2. - Hexa numbers

## example

| 2 | 1 | 0 |  
| -- | -- | -- |  
|16^2 | 16^1 | 16^0 |  
| 1 | 2 | 3 |  
| 256 | 32 | 3 |  
|total=391|
 
 2. **Floating numbers**  
  **Example**
  45.45,57.4,54.4,45.4,0.001  

 3. **Strings**
 - Double Quotes  ("")
 - Single Quotes    ('')
 - Escape Quotes    ('\')

 4. **Boolean**
  - True -> 1
  - False -> 0
