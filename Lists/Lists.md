# List
**len** is a functions which is used to find the length of the words in a given index
### Example 
```console 
$ countries = ("USA","Canada","India")
len[countries] 
output : 3
````
**del** is a function which is used to delete a word from the given data   
### Example 
```console
$ countries = ("USA","Canada","India")
del countries[1]
output : "USA", "India"
